# Product Overview

## Product Purpose

`chroma-sync` は、Adobe Illustrator（.ai）やAdobe Photoshop（.psd）ファイルをJPEG形式に変換する際に発生する色ずれ問題を解決するアプリケーションです。

従来のJPEG変換では、カラープロファイルの不一致やカラースペース変換によって色味が変化してしまうという問題がありました。`chroma-sync` はこの問題を自動的に検出・修正し、元の色味を忠実に再現しながら、ユーザーが指定したファイルサイズまで圧縮します。

ローカル環境およびWeb上で動作し、デザイナーやクリエイターの変換後再調整作業を不要にします。

## Target Users

- **デザイナー・クリエイター**: ai/psdファイルをJPEGに変換してクライアントや印刷会社に納品する必要があるユーザー
- **DTPオペレーター**: 大量の印刷データをWeb用JPEGに変換するプロフェッショナルユーザー
- **Webデザイナー**: 高品質な画像をWeb最適化サイズに変換したいユーザー

**主な悩み・ペインポイント:**
- JPEG変換後に色味が変わり、再調整が必要になる
- ファイルサイズと品質のバランスを手動で調整するのが手間
- 大量ファイルの変換作業が時間がかかる

## Key Features

1. **色味保持変換**: ai/psdファイルをJPEGに変換する際、カラープロファイルを維持して色ずれを防ぐ
2. **自動色味修正**: 変換後に色ずれが発生した場合、差分を検出して自動的に元の色味に戻す
3. **圧縮率指定**: ユーザーが目標ファイルサイズまたは圧縮率を指定できる
4. **ローカル・Web両対応**: Docker環境でのローカル実行とWebアプリとしての利用が可能
5. **バッチ処理**: 複数ファイルの一括変換対応

## Business Objectives

- デザイナーの変換後再調整作業をゼロにする
- 変換作業にかかる時間を大幅に削減する
- 高品質なJPEG出力を自動で実現する

## Success Metrics

- **色差 (ΔE)**: 変換後の色差が ΔE < 2.0 以内（人間が知覚しにくいレベル）
- **変換成功率**: 色味の許容範囲内での変換成功率 95% 以上
- **処理速度**: A4サイズの1ファイルを30秒以内に変換完了

## Product Principles

1. **色の忠実性優先**: ファイルサイズよりも色の正確さを優先する
2. **自動化**: ユーザーが手動で調整する必要なく、自動で最適な結果を提供する
3. **シンプルな操作性**: 技術的な知識がなくても直感的に使える

## Monitoring & Visibility

- **ダッシュボード**: Web UIで変換の進捗・結果をリアルタイム表示
- **色差表示**: 変換前後の色差（ΔE値）をビジュアルで確認可能
- **変換ログ**: 各ファイルの変換結果・修正履歴を記録

## Future Vision

### Potential Enhancements
- **クラウドストレージ連携**: Google Drive、Dropboxからのファイル直接変換
- **API提供**: 外部サービスから利用できるREST API
